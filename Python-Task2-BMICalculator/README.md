# BMI Calculator

## Oasis Infobyte Internship

**Track:** Python Programming

**Task:** Task 2 - BMI Calculator

## Description

This project is a Python-based BMI Calculator developed as part of the Oasis Infobyte Python Programming Internship. It calculates the Body Mass Index (BMI) using the user's height and weight and classifies the result into standard health categories.

## Technologies Used

- Python

## Features

- Accepts weight and height as input
- Calculates BMI
- Displays BMI rounded to 2 decimal places
- Classifies BMI as Underweight, Normal, Overweight, or Obese
- Validates user input
